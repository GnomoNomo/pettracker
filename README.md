# Pet Tracker

Will track times a pet was fed.

# UI

- List of events consisting of:
	- Icon
	- Event Name
	- Event Time
	- Long hold for Edit/Delete
- Buttons for adding events
	- Pee
	- Poop
	- Food
- Popup for adding/editing events
	- Image of event
	- Confirm button
	- Cancel
	- Notes
	- Set time
